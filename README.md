# ParkEase Frontend

# Project Description
ParkEase is a frontend web application developed to support parking management operations.
The system allows attendants to register vehicles, sign them out, and calculate parking
fees, while administrators can view parking reports. This project was built using HTML,
CSS, and JavaScript as part of the Refactory CSE Frontend assignment.

## Pages
- **index.html** – Login page for system users
- **dashboard.html** – Attendant dashboard after successful login
- **register.html** – Vehicle registration form
- **Registeruser.html** – User registration page with role selection
- **signout.html** – Vehicle sign-out and parking fee calculation page
- **reports.html** – Admin reports page
- **tyre.html** – Tyre information page

## Features Implemented
- Login page with role-based access
- Attendant dashboard
- Vehicle registration form
- User registration with role selection
- Vehicle sign-out with parking fee calculation
- Admin reports page
- Client-side form validation using JavaScript

## Validation Rules Applied
- All required fields must be filled
- Number plate format validation
- Phone number validation
- Role selection validation during user registration

## Known Issues / Assumptions
- No backend or database integration
- Data is not persisted after page refresh
- Navigation is frontend-based only
